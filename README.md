# CapsLock Extension 🚀
A high-performance system enhancement tool built with **AutoHotkey v2**. This script transforms the underutilized `CapsLock` key into a versatile **Super-Modifier (Mod-key)**, streamlining text navigation, window management, and introducing a unique **Clipboard-to-File** workflow. It features **encrypted binary persistent storage** for clipboard history to protect your privacy, all without losing the native CapsLock toggle functionality.

## ✨ Key Features
### 📋 Advanced Clipboard Management (With Encrypted Storage)
*   **Paste as File (`CapsLock + V`)**: Instantly convert the last manually copied text into a `.txt` file and paste it (ideal for sending code/text as files in IM apps or File Explorer). The original clipboard content is automatically restored after 10 seconds to avoid disrupting your workflow.
*   **Custom History Menu (`CapsLock + Shift + V`)**: A lightweight, non-blocking clipboard manager with rich capabilities:
    *   **Smart Quick Menu**: Displays up to 15 most recent entries (configurable via `MAX_VISIBLE_MENU`). Click "View full history" to open the complete management GUI.
    *   **Full History GUI**: A resizable window with a multi-select ListView, supporting:
        *   Batch paste multiple selected items as individual `.txt` files with sequential execution.
        *   Full content preview in a resizable read-only window.
        *   Single or batch deletion of history entries.
        *   One-click select/deselect all entries.
    *   **Encrypted Persistent Storage**: Clipboard history is saved to `ClipHistory.bin` in the script directory, using XOR symmetric encryption for binary storage. The history file cannot be read as plaintext directly, fully protecting your copied sensitive content, and survives script reloads and system restarts.
*   **Intelligent Clipboard Locking**: The script tracks and remembers your last manual `Ctrl+C` action via the `LastManualClipboard` mechanism, ensuring `CapsLock + V` always uses your intended content, and avoids polluting the history with temporary files generated by the script itself.

### ⌨️ Productivity Text Navigation
*   **Vim-like Home Row Movement**: Navigate through text with Home/End jumps and word-wise movement, no need to move your hands away from the home row.
*   **Enhanced Deletion**: One-click word deletion (left/right) and full line wipe with a single keystroke.
*   **Quick Selection**: Instantly select the entire current word with a single shortcut.

### 🖼️ Window Control & Real-time Transparency Adjustment
*   **Dynamic Opacity Tuning**: Adjust the transparency of the active window in real time with `CapsLock` + Mouse Buttons.
*   **One-click Stealth Mode**: Toggle the active window between 10% and 100% opacity instantly with `CapsLock + Middle Mouse Button`.
*   **Quick Window State Control**: Maximize/restore, minimize windows with numeric shortcuts, and switch browser tabs with one keystroke.

## ⌨️ Hotkey Reference
### Core Function & Clipboard
| Combo (Hold `CapsLock` +) | Action |
| :--- | :--- |
| Double Tap `CapsLock` | Toggle native CapsLock state (On/Off) |
| `V` | Paste last manually copied text as a **.txt file** |
| `Shift + V` | Open **encrypted clipboard history menu** |

### Text Editing & Navigation
| Combo (Hold `CapsLock` +) | Action |
| :--- | :--- |
| `↑` / `↓` | Jump to line **Home** / **End** |
| `←` / `→` | Jump **left/right by word** (Ctrl + ←/→) |
| `Shift + ↑` / `Shift + ↓` | Select text from cursor to line Home/End |
| `Shift + ←` / `Shift + →` | Select text left/right by word |
| `Space` | Select the entire current word |
| `a` / `d` | **Backspace** / **Delete** |
| `Shift + a` / `Shift + d` | Delete word to the left / Delete word to the right |
| `Backspace` / `Delete` | Delete the entire current line |
| `Q` / `E` | Switch browser tabs (Ctrl + PgUp / PgDn) |

### Window Management
| Combo (Hold `CapsLock` +) | Action |
| :--- | :--- |
| `8` / `Numpad 8` | Toggle window Maximize / Restore |
| `2` / `Numpad 2` | Minimize active window |
| Left Mouse Button (Hold) | Increase active window opacity |
| Right Mouse Button (Hold) | Decrease active window opacity |
| Middle Mouse Button | Toggle window between 10% and 100% opacity |

*Note: The `+` symbol denotes the `Shift` key (e.g., `Shift + a` means `CapsLock + Shift + a`).*

## 🚀 Installation
1.  **Prerequisite**: Install [AutoHotkey v2.0+](https://www.autohotkey.com/) (the script is built exclusively for AHK v2).
2.  **Download**: Clone this repository or download the `CapsLock_Extension.ahk` script file.
3.  **Run**: Double-click the script file to launch it (the script will run in the background with a tray icon).
4.  **Auto-Start Setup**: Right-click the tray icon and select **"Load on start up"** to enable automatic launch on system boot.
    *   This operation modifies the Windows Registry at `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`.
    *   A checkmark will appear next to the menu item, and a success message will confirm the setting is applied.

## 🛠️ Technical Details
*   **Encrypted Binary Persistence**: The clipboard history is stored in a compact binary file `ClipHistory.bin`, using XOR symmetric encryption for the text payload. The `CryptBuffer` function performs byte-level encryption/decryption during write/read operations, ensuring the stored content cannot be viewed as plaintext directly. The encryption key is customizable via the `ENCRYPT_KEY` global variable, and the script maintains backward compatibility with old plaintext history files for a smooth transition.
*   **Auto Self-Cleaning**: Temporary `.txt` files generated by the "Paste as File" feature are automatically deleted after 10 seconds, keeping your system's Temp folder clean with no residual files.
*   **Configurable History Limits**: The clipboard history supports up to **10,000 entries** (defined by `MaxHistory := 10000`), following a FIFO (First-In-First-Out) rule. The quick menu displays a maximum of **15 items** (`MAX_VISIBLE_MENU := 15`) for optimal readability and performance.
*   **Intelligent Clipboard Isolation**: The script uses the `LastManualClipboard` global variable to distinguish between user-initiated manual copy operations and temporary content generated by the script itself. This ensures the "Paste as File" function always uses the user's intended content, and avoids polluting the history with invalid temporary data.
*   **Non-Blocking Menu System**: The history menu uses function-based callbacks (`ActionPickerHandler`) instead of simulated keystrokes, eliminating input conflicts and interference with IME (Input Method Editor) input.
*   **Adaptive Responsive GUI**: Both the full history window and the preview window use `OnEvent("Size")` event listeners, ensuring all UI controls automatically adjust and stay perfectly aligned when the window is resized.
*   **Native CapsLock Compatibility**: The double-tap CapsLock toggle logic preserves the native key functionality, with a 50-300ms time window to avoid accidental triggers, while still allowing the key to be used as a modifier when held down.
